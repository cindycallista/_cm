#  Homework Projects

## HW 1: Calculus & Numerical Approximation
[hw1.py](https://github.com/cindycallista/_cm/blob/main/hw1.py)
* **Derivative:** Approximated using difference quotients with $h = 0.00001$.
* **Integral:** Riemann sum approximation for area calculation.
* **Fundamental Theorem:** Verified relationship between accumulated area and original functions.

## HW 2: Quadratic Equations
[hw2.py](https://github.com/cindycallista/_cm/blob/main/hw2.py)
* **Standard Formula:** Implementation of the quadratic formula for $ax^2 + bx + c = 0$.
* **Complex Support:** Full handling of complex roots using the discriminant.

## HW 3: Cubic Equations
[hw3.py](https://github.com/cindycallista/_cm/blob/main/hw3.py)
* **Normalization:** Transforms equations to monic form.
* **Cardano’s Method:** Uses depression ($x = t - b/3$) and complex cube roots to solve for $x$.

## HW 4: Polynomial Roots via Linear Algebra
[hw4.py](https://github.com/cindycallista/_cm/blob/main/hw4.py)
* **Companion Matrix:** Constructs a matrix where eigenvalues correspond to polynomial roots.
* **Linear Algebra Approach:** avoids iterative root-finding using `np.linalg.eigvals`.

## HW 5: Rational Numbers Class
[hw5.py](https://github.com/cindycallista/_cm/tree/main/hw5)
* **Exact Arithmetic:** Custom class to avoid floating-point errors.
* **Simplification:** Automatic reduction using Greatest Common Divisor (GCD).
* **Overloading:** Implemented `+`, `*`, and `==` operators.

## HW 6: Computational Geometry Engine
[hw6.py](https://github.com/cindycallista/_cm/blob/main/hw6.py)
* **Geometry Classes:** Point, Line, Circle, and Triangle primitives.
* **Algorithms:** Shoelace Formula for area and Cramer’s Rule for line intersections.

## HW 7: Statistical Hypothesis Testing
[hw7.py](https://github.com/cindycallista/_cm/blob/main/hw7.py)
* **Parametric Tests:** Manual implementation of Z-Test and T-Tests.
* **Variance Analysis:** Pooled variance calculation for independent samples.

## HW 8: Information Theory & Coding
[hw8.py](https://github.com/cindycallista/_cm/blob/main/hw8.py)
* **Entropy:** Calculation of Shannon Entropy and KL-Divergence.
* **Hamming Code (7,4):** Implementation of Generator/Parity matrices for error correction.

## HW 9: Linear Algebra & Decompositions
[hw9.py](https://github.com/cindycallista/_cm/blob/main/hw9.py)
* **LU & SVD:** Efficient matrix decompositions for determinants and singular values.
* **PCA:** Dimensionality reduction via covariance matrix and eigen decomposition.

## HW 10: Signal Processing (DFT)
[hw10.py](https://github.com/cindycallista/_cm/blob/main/hw10.py)
* **Fourier Transform:** First-principles implementation of DFT/IDFT.
* **Summation Logic:** Uses Euler’s formula without external FFT libraries.

## HW 11: ODE Solver
[hw11.py](https://github.com/cindycallista/_cm/blob/main/hw11.py)
* **Symbolic Logic:** Solves characteristic equations for homogeneous linear ODEs.
* **Multiplicity:** Handles repeated and complex conjugate roots with automated string formatting.
